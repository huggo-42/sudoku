# Sudoku FOSS
Sudoku game ad-free, just play.

You can:
- Play Sudoku!
- Create new boards.
- Erase selected squere.

Features:
- Smart highlights (I think they make the game too easy...)
  - selected line, column, quare
  - error indicator
